Overview

Neural netwoks are a powerful, nonparametric, supervised machine learning algorithm; convolutional neural networks are a specialized class which develop features and classify images specifically. A recent surge in the use of Convolutional Neural Networks for image classification has lead to lots of data and network parameters being released. We experiments studies done by engineers at Google who reversed the optimization problem in the network and were able to achieve novel imaging effects.

Neural Networks as a Classifier

The neural network process one layer at a time, and it learns by comparing the classification of each layer with the known actual classification. The errors from the initial classification of the first record is fed back into the network, and used to modify the networks algorithm for further iterations.
